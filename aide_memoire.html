Lois de base

$U=R\,I$ $P=U I$ ↳ $=R I^2=\dfrac{U^2}{R}$ (via Ohm) $W=P\,t=U I t$ $R=\rho\,\dfrac{L}{S}$ (1 mm²$=10^{-6}$m²)

Coût : énergie(kWh)=P(W)·t(h)/1000, ×prix.

Résistances

Série : $R_{eq}=R_1+R_2+\dots$ Parallèle : $\dfrac1{R_{eq}}=\dfrac1{R_1}+\dfrac1{R_2}+\dots$ ↳ 2 résist. : $R_{eq}=\dfrac{R_1R_2}{R_1+R_2}$ (via 1/R_eq)

Série = même courant. Parallèle = même tension. On additionne les séries, on combine en 1/R les parallèles.

Kirchhoff & diviseurs

Maille : $\sum U=0$ Nœud : $\sum I_{ent}=\sum I_{sort}$ ↳ Div. tension : $U_{R2}=U\dfrac{R_2}{R_1+R_2}$ (série + Ohm) ↳ Div. courant : $I_1=I\dfrac{R_2}{R_1+R_2}$ (// + Ohm)

Diviseur de courant : l'autre R au numérateur.

Signaux sinusoïdaux

$v(t)=V_{max}\sin(2\pi f t)$ $\omega=2\pi f \qquad f=\dfrac1T$ $V_{eff}=\dfrac{V_{crête}}{\sqrt2} \quad V_{crête}=V_{eff}\sqrt2$ $V_{pp}=2\,V_{crête}$

Moyenne d'une sinus pure = 0 (sauf après redressement).

Transistor — relations

$I_E=I_C+I_B$ $\beta_{CC}=\dfrac{I_C}{I_B}\Rightarrow I_C=\beta I_B$ ↳ $\alpha=\dfrac{I_C}{I_E}\approx0{,}99$ ($=\beta/(\beta+1)$) $V_{BE}\approx0{,}7\,$V

3 zones : blocage ($I_C\!=\!0$), active ($I_C\!=\!\beta I_B$), saturation ($V_{CE}\!\approx\!0$).

Droite de charge / point Q

Saturation : $I_{Csat}=\dfrac{V_{CC}}{R_C}$ ($V_{CE}=0$) Blocage : $V_{CE}=V_{CC}$ ($I_C=0$) $V_{CE}=V_{CC}-R_C I_C$ Base : $I_B=\dfrac{V_{BB}-0{,}7}{R_B}$

↳ $I_{Csat},I_B,V_{CE}$ se déduisent (Ohm + maille + déf β). Si $\beta I_B>I_{Csat}$ → saturé : $I_C=I_{Csat}$, le β ne s'applique plus.

Polarisation pont diviseur (+ R_E)

$V_B=V_{CC}\dfrac{R_2}{R_1+R_2}$ $V_E=V_B-0{,}7$ $I_C\approx I_E=\dfrac{V_E}{R_E}$ $V_C=V_{CC}-R_C I_C \quad V_{CE}=V_C-V_E$ $I_{Csat}=\dfrac{V_{CC}}{R_C+R_E}$

Transistor en interrupteur

$I_{Csat}=\dfrac{V_{CC}-V_{CEsat}}{R_C}$ $I_{Bmin}=\dfrac{I_{Csat}}{\beta}$ $R_B=\dfrac{V_{BB}-0{,}7}{I_B}$

↳ tout via Ohm ($V_{CE}=0$) + déf β. $R_B$ trouvé = max pour juste saturer ; prendre + petit pour garantir.

Semi-conducteurs (QCM)

  • Le + utilisé : silicium (4 e⁻ valence)
  • Électrons libres : bande de conduction
  • Noyau : protons + neutrons
  • Type N : impureté 5 e⁻ (électrons) · Type P : 3 e⁻ (trous)
  • Polar. directe PN : + sur P (anode), − sur N (cathode)
  • LED : recombiner e⁻/trous → lumière

⚠️ Pièges fréquents

  • Convertir les unités (mA, kΩ, µF, mm²)
  • Pont : 1,4 V et fréquence ×2
  • LED ≠ 0,7 V (≈ 2 V) + toujours une R
  • Zéner : vérifier $I_{Zmin}
  • Transistor saturé : β ne s'applique plus
  • Diviseur tension : invalide si charge dérive le courant

Réflexes méthode

  • Diode : supposer passante (0,7) → vérifier $I_D>0$.
  • Redresseur : V_crête → −chute → ondulation ($f{=}2f$) → V_moy.
  • Zéner : $I_R,I_{ch},I_Z$ → vérifier seuils.
  • Transistor : comparer $\beta I_B$ et $I_{Csat}$ → actif ou saturé.
  • R_total : additionner les R en série, combiner les // ; diodes/LED = chutes (jamais des R).
